# CoreData
В данном задании необходимо было использовать разные инструменты для хранения данных, такие как UserDefaults, Realm, CoreData.
На первом экране по нажатию кнопки в UserDefaults сохраняются введенные данные и при следующим запуске оторбражаются последние сохраненные данные.
На втором и третьем экране приложение To Do для создани заметок и дел. В первом варианте хранение данных происходит через Realm, во втором варианте через CoreData.
На последнем экране прогноз погоды, который до обновления данных показывает последние сохраненные данные.
Погода в данный момент сохраняется в UserDefaults, а прогноз на 5 дней сохраняется в Realm.
